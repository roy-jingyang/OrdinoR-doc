orgminer.ResourceProfiler.raw_profiler module¶

This module contains the implementation of methods for profiling resources using information directly accessible in a resource log.

orgminer.ResourceProfiler.raw_profiler.count_execution_frequency(rl, scale=None)¶

Build resource profiles based on how frequently resources originated events of execution modes.

Each column in the result profiles corresponds with an execution mode captured in the given resource log.

Parameters

rl : DataFrame

A resource log.

scale : {None, ‘normalize’, log’}, optional, default None

Options for deciding how to scale the values of frequency counting. Could be one of the following:

  • None, no scaling will be performed.

  • 'normalize', scale the frequency values by the total count of executions by each resource.

  • 'log', scale the frequency values by logarithm.

Returns

DataFrame

The constructed resource profiles.

Raises

ValueError

If the specified option for scaling is invalid.
