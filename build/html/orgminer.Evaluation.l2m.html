orgminer.Evaluation.l2m.conformance.fitness(rl, om)¶

Calculate the fitness of an organizational model against a given resource log.

Parameters

rl : DataFrame

A resource log.

om : OrganizationalModel

The discovered organizational model.

Returns

float

The result fitness value.

Notes

A resource log instead of an event log is used here, however this is valid since a resource log is (implicitly) one-to-one corresponded with an event log.
